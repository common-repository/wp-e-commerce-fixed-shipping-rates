== Description ==

Enables free control over fixed rate shipping options, like:

*   In Store - $0
*   Regular - $5
*   2nd Day - $10
*   etc..

== Installation ==

1. Upload files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Store Settings ->Shipping, Fixed Rate will appear in Internal Shipping Calculators list.
